Bias Detection and Correction Evaluation for Swahili

Project: JuaKazi Gender Bias Detection & Correction System
Language: Swahili (SW)
Evaluation Date: 12 January 2026

1. Dataset Description

The evaluation was conducted on a manually curated Swahili dataset consisting of 63 sentences. Of these, 31 instances were annotated as containing gender bias. Bias categories include pronoun assumption, pronoun generic, and occupation-related bias.

6. Technical Interpretation

Results indicate that the JuaKazi correction pipeline achieves complete bias neutralization for all detected Swahili bias categories, while preserving semantic integrity. Token-level analysis confirms that edits are minimal and targeted, ensuring content fidelity.

Detection recall is lower for certain categories, such as occupation-related bias, indicating the need for improved detection coverage without sacrificing precision.
