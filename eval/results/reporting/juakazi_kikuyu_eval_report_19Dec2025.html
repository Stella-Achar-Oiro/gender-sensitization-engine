Bias Detection and Correction Evaluation for Kikuyu

Project: JuaKazi Gender Bias Detection & Correction System
Language: Kikuyu (KI)
Evaluation Date: 19 December 2025

1. Dataset Description

The evaluation was conducted on a manually curated Kikuyu dataset consisting of 33 sentences. Of these, 18 instances were annotated as containing gender bias. Bias categories include pronoun assumption and occupation-related bias.

6. Technical Interpretation

Results indicate that the JuaKazi correction pipeline achieves complete bias neutralization for detected Kikuyu pronoun-based bias, while preserving semantic content. The correction strategy relies on minimal lexical substitution, which is particularly suitable for low-resource and morphologically rich languages.

Detection recall remains the primary limitation, especially for occupation-based bias categories, suggesting future work should focus on expanding detection coverage without compromising precision.
