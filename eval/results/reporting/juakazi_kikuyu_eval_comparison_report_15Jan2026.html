Comparative Bias Correction Evaluation – Kikuyu (KI)

Project: JuaKazi Gender Bias Detection & Correction System
Languages: Kikuyu (KI)
Comparison Scope: Small-scale vs Large-scale Evaluation

1. Evaluation Scope Overview

This report compares two evaluation runs of the JuaKazi bias detection and correction system for Kikuyu. The first evaluation was conducted on a small, manually curated dataset, while the second represents a large-scale evaluation intended to stress-test detection coverage, correction reliability, and semantic preservation.

Detection performance reflects the system’s ability to identify biased text prior to any intervention. Both runs were evaluated under a precision-first operating point.

4. Semantic Preservation Comparison

Semantic preservation metrics assess how well meaning is retained after bias neutralization.

Metric Run A Run B
BLEU 0.854 0.703
ROUGE-L 0.823 0.824
Token Overlap 0.783 0.824
Edit Similarity 0.783 0.824
Composite 0.816 0.787

At scale, BLEU decreases due to increased lexical diversity, while ROUGE-L and token overlap remain stable or improve. This suggests that corrections remain structurally faithful, even when surface-level wording varies.

5. Areas of Success

The following strengths are consistently observed across both evaluations:

  • Zero false positives across all datasets, ensuring safe deployment.
  • Perfect correction reliability once bias is detected.
  • High semantic preservation, even at large scale.
  • Strong pronoun-bias handling, with high pre-F1 and full neutralization.
  • Scalable correction logic unaffected by dataset size.

6. Areas Requiring Improvement

Despite strong correction performance, several limitations remain:

  • Detection recall remains below 0.50, limiting end-to-end impact.
  • Occupation bias detection shows consistently low pre-F1.
  • HarmonicScore ceiling is constrained by detection rather than correction.
  • BLEU sensitivity at scale suggests a need for embedding-based semantic metrics.

Future work should prioritize expanding detection coverage—particularly for occupation-related bias—while preserving the system’s conservative precision profile.
